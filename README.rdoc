# Веб-приложение для статистического анализа наборов данных

Пользователи могут CRUD наборы данных — двухстрочные массивы double. Для наборов в системе есть анализаторы: матожидание, дисперсия, построние гистограмы, прогнозирование… Анализаторы принимают на вход набор, а на выходе дают данные разного формата: график, массив, число.

Администратор может загружать анализаторы, к которым предъявлен ряд требований.

Таким образом, система гибкая с обеих сторон.

Для начала все анализаторы пишутся классами в приложении, потом я их отгружу в базу.
